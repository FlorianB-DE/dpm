DPM - Docker Package Manager

Welcome to DPM, your new way of running programs!

DPM (Docker Package Manager) is a Rust-based command-line tool that allows you to replace locally installed programs with a container running that service. It is designed to make it easier to keep track of your installed programs and dependencies.

When you only need a program once, you don't have to install a lot of dependencies. Once you remove the image from your system, it's gone entirely.

Installation

To install DPM, you need to have Rust installed on your system. You can install Rust by following the instructions on the official Rust website.

Once Rust is installed, you can install DPM by running the following command:

cargo install dpm

Usage

To use DPM, you can run the following command:

dpm <command> [<args>]

Available commands:

  • install [--image=<name>] [--tag=<tag>] - Install a program using a Docker image.
  • list - List installed programs.
  • remove <name> - Remove an installed program.
  • start <name> - Start a program.
  • stop <name> - Stop a running program.
  • update <name> - Update an installed program.
